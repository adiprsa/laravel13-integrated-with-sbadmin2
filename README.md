# Laravel 13 with SB Admin 2  
Inspired from <https://dev.to/raflizocky_/integrate-sb-admin-2-in-laravel-40a3>  
## How To Run  
1. `npm install`  
2. `php artisan migrate`  
3. `php artisan serve`  
  
## Features  
1. Register Login Logout, see branch `authentication`  
1. CRUD, see branch `crud`
